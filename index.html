Publications

  1. 3D Gaussian Splatting Compression with Object Scalability
    ECCV 2026 · European Conference on Computer Vision
    Ruixiang Xue et al. (authors to be finalized after public release)
    ECCV 2026Accepted3DGS Compression
    ProjectPaperCodeBibTeX
    Abstract

    We introduce a framework towards scalable, finer-grained object-level 3D Gaussian Splatting compression. RecastGS reorganizes pretrained 3DGS into a region-aware layered hierarchy for object-level and ROI-adaptive quality control. LayeredCGS then compresses position with a lightweight point cloud codec and attributes with cross-layer context modeling, producing truncatable bitstreams for low-latency preview and progressive refinement.

    Accepted, not yet on arXiv
  2. A Versatile Point Cloud Compressor Using Universal Multiscale Conditional Coding-Part I Geometry
    TPAMI · IEEE Transactions on Pattern Analysis and Machine Intelligence
    Jianqiang Wang, Ruixiang Xue, Jiaxin Li, Dandan Ding, Yi Lin, Zhan Ma
    SCI Q1CCF-AIF 18.6Co-first author
    ProjectPaperCodeBibTeX
    Abstract

    Proposed a universal multiscale conditional point cloud geometry coding framework that supports lossy/lossless, static/dynamic, and diverse point cloud data, outperforming MPEG G-PCC, V-PCC, and learning-based baselines with strong compression efficiency.

    SCI Q1, CCF-A, impact factor 18.6, co-first author
  3. NeRI: Implicit Neural Representation of LiDAR Point Cloud Using Range Image Sequence
